🔄 Ciclos

¿Qué son?

Un ciclo es una estructura que permite ejecutar una o varias instrucciones de forma repetitiva hasta que se cumpla una condición específica.

Tipos

  • for
  • while
  • do while

Ejemplo FOR

for(int i=1;i<=5;i++){

    System.out.println(i);

}

Errores comunes

  • Ciclos infinitos.
  • No incrementar variables.
  • Condiciones incorrectas.

Aplicaciones

Videojuegos, sistemas bancarios, inventarios.

🛠 Métodos

Los métodos son partes de un programa que se utilizan para realizar una tarea específica. Permiten reutilizar código varias veces sin necesidad de volver a escribirlo, lo que hace que los programas sean más organizados, fáciles de entender y de mantener. 

public static int sumar(int a,int b){

    return a+b;

}

Ventajas

  • Organizan el código.

  • Facilitan mantenimiento.

📊 Vectores y Matrices

Vector

Un vector es una lista de datos almacenados en una sola variable, donde cada dato tiene una posición específica. 

int[] numeros={10,20,30,40,50};

Matriz

Una matriz es una tabla de datos organizada en filas y columnas, similar a una hoja de cálculo. 

int[][] matriz={
 {1,2,3},
 {4,5,6},
 {7,8,9}
};
